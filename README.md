# OpenCart Manual Testing Project

A complete manual testing project on the OpenCart e-commerce 
demo application, built as a portfolio project to demonstrate 
core QA skills.

---

## Project Overview

| Field | Details |
|---|---|
| Application | OpenCart E-Commerce (demo.opencart.com) |
| Testing Type | Black-box Manual Functional Testing |
| Test Cycle | Cycle 1 — May 2026 |
| Tester | Krishna Singh |
| Tool Used | Microsoft Excel |
| Browser | Google Chrome (Latest) |
| OS | Windows 10 |

---

## Deliverables

| Document | Description |
|---|---|
| Test Plan | Scope, objectives, approach, environment, entry/exit criteria |
| Test Cases | 25 test cases covering positive and negative scenarios |
| Bug Report | 3 bugs logged with severity, priority, steps to reproduce |
